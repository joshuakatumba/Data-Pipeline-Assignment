# Global Patent Intelligence Data Pipeline

This repository contains a full data engineering pipeline for processing, cleaning, storing, and analyzing real-world patent data. It simulates the extraction of data from the USPTO PatentsView APIs and constructs a reproducible SQLite database and dashboard.

## Project Structure

- `data/raw/`: Contains the synthetic raw patent data.
- `data/clean/`: Contains the cleaned CSVs generated by pandas.
- `scripts/`: Python scripts that execute each step of the pipeline.
- `sql/`: SQL schema definitions and analytical queries.
- `reports/`: Exported CSVs and JSON reports.
- `app.py`: Interactive Streamlit dashboard.

## Requirements

Ensure you have Python 3 installed. Install the dependencies using:

```bash
pip install -r requirements.txt
```

## Running the Pipeline

To run the pipeline and generate all reports, execute the scripts sequentially:

1. **Extract Data:** Generates the raw JSON patent records.
   ```bash
   python scripts/1_extract_data.py
   ```

2. **Clean Data:** Cleans the messy JSON data into normalized pandas DataFrames and exports them to CSVs.
   ```bash
   python scripts/2_clean_data.py
   ```

3. **Load to Database:** Initializes `patents.db` using `schema.sql` and loads the cleaned data into relational tables.
   ```bash
   python scripts/3_load_to_sql.py
   ```

4. **Analyze & Report:** Runs complex analytical SQL queries (Q1-Q7) and generates terminal outputs, CSV extracts, and a JSON report.
   ```bash
   python scripts/4_analyze_and_report.py
   ```

## View the Dashboard (Extra Marks)

You can launch the interactive Streamlit dashboard to view data visualizations (Top Inventors, Top Companies, and Patent Filings Over Time).

```bash
streamlit run app.py
```

## SQL Queries Implemented
The file `sql/queries.sql` contains:
- **Q1**: Top Inventors (most patents)
- **Q2**: Top Companies (most patents)
- **Q3**: Countries (produce most patents)
- **Q4**: Trends Over Time (patents per year)
- **Q5**: JOIN Query (combining patents, inventors, and companies)
- **Q6**: CTE Query (using WITH to break down complex logic)
- **Q7**: Ranking Query (using window functions to rank inventors)
